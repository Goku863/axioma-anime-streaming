# AnimeForYou - Premium Anime Streaming

A modern, dark-themed anime streaming website with glass morphism effects, smooth animations, and responsive design.

## Features

- **Home Page** - Hero section with featured anime, trending content, and stats
- **Browse/Work** - Anime grid with filters, search, and genre tags
- **Episode/Watch** - Video player with episode list and related content
- **Download** - Download page with quality options
- **About** - Company info, team, and stats
- **Services/Features** - Feature cards with pricing
- **Contact/Support** - FAQ and contact form
- **Login** - Authentication with Google/GitHub
- **Profile** - User profile with watch history
- **Admin Panel** - Dashboard with stats, anime management, and activity feed
- **Automation** - AI automation features

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- JavaScript (Vanilla)
- Google Fonts (Inter, JetBrains Mono)
